Precision Craftmanship

Red Radish Rat is an experimental system that automates design. The aim of the system is to craft design artifacts, such as patterns, apps and websites, with great precision. Each artifact is unique, yet precicely tuned and produced for each end use.

Straight from the Factory

Automation allows for design artifacts to be produced on the fly, as required. With minimal human intervention, a product can be delivered with very short turn around times.

Expertly Crafted

Each item is expertly crafted with the greatest attention to detail. We produce unique products, individual and with character.

Driven by Code

Behind the scenes most of the operation is driven by code. Code never gets tired. Never.
